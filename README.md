# Regular Languages are Closed Under Complement

This is a final programming project for Theory of Computation. The purpose of this project is to provide a visual demonstration of how every regular language can have a complement DFA that is the complement.

- Supports custom input strings and alphabets  
- Visualizes DFA transitions
